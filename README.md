# 📚 Library Management System

A simple **web-based Library Management System** developed for college project purposes.  
This application helps manage books in a library by allowing users to add, issue, return, search, and delete books.

---

## 🚀 Features

- 🔐 Login System
- ➕ Add New Books
- 📖 View Book List
- 🔍 Search Books
- 👨‍🎓 Issue Books to Students
- 🔁 Return Books
- 🗑 Delete Books
- 📜 Issue History Tracking
- 💾 Data stored using Browser Local Storage
- 🎨 Simple and Clean User Interface

---

## 🛠 Technologies Used

- HTML
- CSS
- JavaScript
- Local Storage (for storing data)
